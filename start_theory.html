Что делает эта программа¶

В этой программе показываются две техники: синтаксис для создания объектов триангуляции и некоторые элементы циклов для «прохождения» по всем ячейкам.

Мы создаём две сетки. Первая - квадрат, который разбивается на множество квадратов (не самая интересная задача, но эта сетка является начальной в других задачах); вторая - кольцевая область, которая «детализируется» при приближении к внутреннему краю.

Благодаря данной программе, мы узнаем, что должна иметь каждая программа, использующая метод конечных элементов:

  • объект типа Triangulation для сетки;

  • вызов функции GridGenerator для создания сетки;

  • циклы для перебора ячеек с использованием итераторов (итераторы являются обобщением указателей и часто используются в стандартных библиотеках C++);
